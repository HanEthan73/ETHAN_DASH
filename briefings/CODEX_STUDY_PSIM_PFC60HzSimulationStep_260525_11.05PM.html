PSIM 60Hz PFC 시뮬레이션 3분 걸릴 때 Simulation Step으로 빠르게 돌리는 법

생성 2026-05-25 23:05 KST | 질문: “60Hz PFC가 약 3분 걸리는데 simulation step 이용하면 빨라진다는데 어떻게?”
핵심
스위칭 상세 모델과 라인주기 모델을 분리
가속 1순위
Max step / print step 분리
가속 2순위
초기조건 저장 후 재시작
금지
스위칭 edge를 놓치는 큰 step

1. 먼저 결론

60Hz PFC 시뮬레이션이 느린 이유는 60Hz 한 주기를 보려면 수십 ms를 돌려야 하는데, 회로는 수십~수백 kHz 스위칭 edge까지 따라가야 하기 때문입니다. 빠르게 하려면 simulation step을 무작정 크게 하는 게 아니라 목적별로 step을 나눠야 합니다. “스위칭 파형 검증”은 작은 step, “라인주기 PF/THD/전압 루프 확인”은 평균모델 또는 큰 print step로 갑니다.
목적권장 방식왜 빠른가주의점
게이트/전류 ripple/diode recovery 확인스위칭 상세 모델
작은 time step 유지		정확도 우선. edge를 놓치지 않음느림. 긴 60Hz transient 전체를 이 방식으로 돌리면 시간 낭비
60Hz 입력전류 모양, PF, Vout settlingaverage model / simplified switch스위칭 edge를 계산하지 않아 수십 배 빨라짐스위칭 ripple, SR/diode 세부 손실은 못 봄
정상상태 주변만 상세 확인초기조건 저장 후 restart기동 100~300ms를 매번 다시 계산하지 않음초기조건 파일이 바뀐 회로 조건과 맞아야 함
파형 저장/화면 출력 줄이기print step 크게, internal step 작게계산은 촘촘히, 저장 포인트는 듬성듬성print step을 너무 크게 하면 보고 싶은 ripple이 안 보임

2. Simulation Step을 어떻게 잡나

파라미터의미추천 시작값판단 기준
Time Step / Max Step내부 적분 계산의 최대 간격스위칭 상세: Tsw/100 ~ Tsw/200
예: 100kHz면 50~100ns		인덕터 전류 ripple과 MOSFET edge가 매끄럽게 보여야 함
Print Step결과 저장 간격라인주기 관찰: 5~20us
스위칭 ripple 관찰: 50~200ns		저장 파일 크기와 그래프 표시 속도를 줄이는 핵심
Total Time전체 시뮬 시간기동 확인: 100~300ms
정상상태 상세: 마지막 1~3ms만		기동과 정상상태를 한 파일에서 계속 반복하지 말 것
Save / Load Initial Condition정상상태 근처를 저장해 재시작첫 run에서 steady-state 저장, 이후 parameter sweep은 loadPFC 전압루프가 안정된 지점에서 저장해야 함

4. 바로 적용 체크리스트

순서작업설정 예시성공 판단
1현재 3분짜리 run에서 저장 파형 줄이기불필요한 scope/probe 출력 제거, print step 10us 이상결과 파일 크기와 표시 시간이 줄어듦
2입력 60Hz 3~5주기만 평균모델로 안정화스위치 대신 average PFC block 또는 behavioral switch 사용Vout, input current envelope 확인 가능
3정상상태 저장Save initial condition / snapshot 기능 사용다음 run이 기동 없이 바로 정상상태 근처에서 시작
4마지막 1~3ms만 상세 스위칭 모델로 재실행max step = Tsw/100, print step = Tsw/50부터 시작스위칭 ripple, 전류 피크, ZVS/ZCS 판단 가능
5파라미터 sweep은 초기조건 기반 반복Vin, Load, L, C를 바꿔도 기동시간을 매번 계산하지 않음반복 run 시간이 분 단위에서 초 단위로 내려감

최종 요약

대표님 케이스에서 “simulation step으로 빠르게”의 정답은 step을 크게 하나로 잡는 것이 아닙니다. 내부 계산 step, 출력 저장 print step, 평균모델/상세모델, 초기조건 재시작을 분리하는 것입니다. 60Hz PFC 전체 동작은 평균모델로 빠르게 보고, 실제 스위칭 파형은 정상상태 마지막 몇 ms만 상세모델로 보면 됩니다.
